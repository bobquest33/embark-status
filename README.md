# Installation

```npm install status-embark-plugin --save```

# Configuring

Add this config in `embark.json`:

```Json
  "plugins": {
    "status-embark-plugin": {
      "deviceIp": "your-device-ip",
      "whisperIdentity": "dapp-test",
      "name": "MyDapp"
    }
  }
```

then ```embark run```
